---
description: "Better Follow-Up Nudges"
---

Follow-up reminders in Slack now show you exactly what's going on: who owes a reply, a snippet of the last message, and a **Mark done** button so you can resolve a thread without leaving Slack.

- Edited draft replies sent from the messaging modal now go out immediately — no extra confirmation step
- Booking links in AI-drafted replies only appear when you're actually scheduling, not as a generic call to action
- Reply memory produces more actionable writing-style summaries, so drafts match your tone more consistently
